# Diveni Frontend

## Get Started

1. Install Expo Go on your device.
2. Already have installed node.js on your pc.
3. npm install (on the dir with the project).
4. npx expo start (on the dir with the project).
5. Scan the QR with your device.
6. Enjoy ;)
